Basic concept

  • What is a JavaScript Object?

    The Object is the fundamental building block of the language. While other primitives (like strings, numbers, and booleans) store a single value, objects are used to store keyed collections of various data and more complex entities.

    At its core, an object is a collection of properties. A property is an association between a key (or name) and a value.

    1. Keys: Must be Strings or Symbols.
    2. Values: Can be any data type (Numbers, Strings, Arrays, Functions, or even other Objects).

    When a property's value is a function, we call it a Method. Mental Model: Think of an object like a real-life dictionary or a filing cabinet. The "Key" is the word you look up, and the "Value" is the definition inside.
