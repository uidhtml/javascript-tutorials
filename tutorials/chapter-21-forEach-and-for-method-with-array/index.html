for() and forEach() with an Array

What is a forEach loop?

In JavaScript, forEach() is a built-in method available on Array and other iterable objects like Set and Map. It is used to iterate over the elements of the collection and execute a provided callback function once for each element.

Key characteristics of forEach():
  • Iteration
  • Callback Function
  • Callback Arguments:
    • lcurrentValue: The current element being processed.
    • index (optional): The index of the current element.
    • (optional): The array forEach() was called upon.
  • No Return Value
  • Immutability (of the original array): It does not modify the original array unless the callback function explicitly performs modifications.
  • No Early Exit: Unlike traditional for loops, you cannot use break or continue to exit or skip iterations within a forEach() loop.
