What is a for...of loop?

The for...of statement creates a loop that iterates over iterable objects such as arrays, strings, maps, and sets. It provides a simple, clean way to loop through the values of an iterable, rather than the indices or keys.

Syntax:
for (const element of iterable) {
  // code to be executed for each element
}
