String overview with template literals (ES6)

What is string?

In JavaScript, a string is a primitive data type used to represent and work with a sequence of characters. It's one of the fundamental building blocks for any web application.

How many ways we represent string?
  • Double quote ("...")
  • Single quote ('...')
  • Multiline quote (`...`)
  • Escape charecter quote ('..\'\'.')
  • Template iterals (ES6) (`${variable}...`)
