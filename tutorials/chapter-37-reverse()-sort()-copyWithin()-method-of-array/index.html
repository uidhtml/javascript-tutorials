Javascript tutorials:
reverse(), sort() and copyWithin() method of an array

reverse()

Reverse the array content and mutate the array.

Syntax
Array.reverse();

sort()

Sort the array content and mutate the array.

Syntax
Array.sort(); //Without compareFn
Array.sort((a, b) => a - b); //With compareFn ascending order
Array.sort((a, b) => b - a); //With compareFn descending order

copyWithin()

Copy content of array with in the array and mutate the array.

Syntax
Array.copyWithin(target, start, end); 
/*
target: At which index content will get pasted.
start: Pick the start index of elements from the array
end: Pick the last index range of the elements from the array
*/
