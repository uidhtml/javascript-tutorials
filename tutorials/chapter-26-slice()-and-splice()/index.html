slice() and splice() of array method

.slice()

The slice method extracts a section of an array and returns it as a new array.

Syntax
array.slice(startIndex, endIndex);

.splice()

The splice method funndamentally changes the content of an array by removing, replacing or adding elements in place.

Syntax
array.splice(startIndex, deleteCount, item1,item2,...)
