What is an Array?

In JavaScript, an array is a special type of object used to store an ordered collection of multiple values under a single variable name. These values, known as elements, can be of any data type (numbers, strings, objects, even other arrays) and are stored in a specific sequence.

Ways to create Array
  • let array = [];
  • let array = new Array();
  • let array = new Array().fill()
  • let array = new Array().from()
