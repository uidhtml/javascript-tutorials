Traditional loop (for(), while() and do-while())

What is a loop?

In JavaScript, a loop is a control structure used to repeatedly execute a block of code until a specified condition is met. Loops provide an efficient way to automate repetitive tasks, such as iterating over elements in an array, performing calculations multiple times, or generating dynamic content.

Types of treditional loop
  • for()
  • while()
  • do-while()
Types of loop control statement
  • break
  • continue
