unshift() and shift() of array method

.unshift()

The unshift() method adds one or more elements to the begining of an array.

Syntax
array.unshift(element1, element2, ..., elementN)

.shift()

The shift() method removes the first element from the begining of an array.

Syntax
array.shift()
